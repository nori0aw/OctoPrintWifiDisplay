# OctoPrintWifiDisplay
OctoPrint Wifi Display is a Esp8266 based project that will display the status from mqtt plugin of Octoprint instance and display it on character LCD screen (20X4).


Boards:
 - ESP8266
 - LCD screen (20X4)
